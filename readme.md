### Installation guide ###

#### Glimpse For Windows ####

Move the brushes into `C:\Program Files (x86)\Glimpse Image Editor\Glimpse 0.1.2\share\glimpse\2.0\brushes`.

Restart Glimpse. 

#### GIMP For Windows #### 

Move brushes into `C:\Program Files\GIMP 2\share\gimp\2.0\brushes`.

Restart GIMP.
